# VeriQuest

AI-Powered Trustless Quest & Reputation Protocol on GenLayer

---

## Overview

VeriQuest is a decentralized reputation and campaign evaluation protocol built on GenLayer.

The protocol enables organizations, DAOs, communities and brands to create content challenges and automatically evaluate participant submissions using AI-powered consensus.

Instead of relying on centralized moderators, VeriQuest uses GenLayer Intelligent Contracts to score content quality, assign reward tiers, update reputation and determine campaign outcomes.

---

## Why VeriQuest?

Traditional quest and bounty platforms depend on human reviewers.

This creates:

- Reviewer bias
- Slow moderation
- High operational costs
- Poor scalability
- Lack of transparency

VeriQuest solves this using AI-powered evaluation directly inside Intelligent Contracts.

## Features

### AI Content Evaluation

Automatically evaluates submissions.

### Reward System

Reward tiers:

- Gold
- Silver
- Rejected

### Reputation Engine

Tracks:

- Reputation Score
- Successful Submissions
- Failed Submissions
- Reward History

### Campaign Decisions

Automatically determines:

- Approved
- Review

Tracks:

- Winning Submission
