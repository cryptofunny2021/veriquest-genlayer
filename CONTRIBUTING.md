# Contributing

Thank you for your interest in VeriQuest.

VeriQuest is an experimental Intelligent Contract built on GenLayer that explores AI-powered content evaluation, reputation systems, and trustless campaign management.

## Ways to Contribute

* Report bugs
* Suggest improvements
* Improve documentation
* Propose new reputation models
* Improve reward distribution logic
* Build front-end integrations

## Development Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Guidelines

* Keep code readable
* Document major changes
* Test contracts before submitting
* Follow GenLayer Intelligent Contract best practices

We welcome feedback and collaboration from the GenLayer community.
