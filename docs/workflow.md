# VeriQuest Workflow

1. Create Quest

Campaign creator creates a new challenge.

2. Submit Content

Contributors submit content.

3. AI Evaluation

GenLayer evaluates content using decentralized AI consensus.

4. Reward Assignment

Content receives:

* Gold
* Silver
* Rejected

5. Reputation Update

Contributor reputation is updated.

6. Campaign Decision

Protocol determines:

* Approved
* Review
* Rejected

7. Winner Selection

Highest scoring submission becomes the winner.
